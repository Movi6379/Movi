# Smart Grid Fault Detection Dashboard

This project integrates **ETAP Short Circuit Analysis** with a **Python-based Dashboard** for real-time monitoring and fault rectification.

## 🚀 Features
- **Fault Identification:** Specific location and type (LLL, LG, LL, LLG).
- **Severity Analysis:** Categorizes faults as Low, Medium, or Critical.
- **Visuals:** Voltage dip profiles and current magnitude graphs.
- **Rectification:** Automated engineering recommendations for grid stability.

## 🛠️ Tech Stack
- **Simulation:** ETAP 
- **Data Processing:** Python (Pandas, NumPy)
- **Visualization:** Plotly
- **UI:** Streamlit
